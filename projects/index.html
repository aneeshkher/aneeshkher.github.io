A list of all my significant academic projects and assignments (to be completed soon)

  • Systems
    • RAM File System (C Programming, FUSE)
      Developed an in-memory file system using the FUSE library. The FUSE library captures all the system calls, and returns them to the user space. I wrote implementations for the basic system calls like read, write, open, close, etc.
    • Non preemptive threading library (C Programming)
      Developed a non preemptive threading library using context based system calls (setcontext, makecontext, swapcontext, getcontext) which supported create, yield, join, and exit thread routines. Supported semaphore routines - init, signal, wait, destroy.
    • Linux Micro shell (C Programming)
      Developed a micro command line shell which accepted all linux commands. Implemented custom built-in functions for a few commands - cd, echo, logout, nice, pwd, setenv, unsetenv, and where. Supported I/O redirection and inter-process communication using the pipe command and duplication of file descriptors.
    • Hot porato game simulation (C Programming)
      Developed a network of computers where each player would play the Hot Potato game. The modified version of the game consisted of a packet being passed around the ring, with a specified number of hops, where each player would append its player id to the packet string. As soon as the number of hops reduces to 0, the player with the packet will send the packet back to the master.
  • Networking
    • Simple FTP using Go-Back-N ARQ (Java. Team of 2)
      Implemented reliable file transfer between two machines using UDP sockets. Achieved reliability using the Go-Back-N ARQ protocol. Used threads for retransmission timer and for packet resend functionality.
    • Peer to Peer using Centralized Index (Java. Team of 2)
      Implemented a simple peer to peer system for downloading RFCs. Maintained a centralized index on a known server which contained information about the location of RFCs on the network.
  • Database Management
    • University Housing Application (Java, SQL. Team of 4)
      A database application which handles all the housing and parking options for students and registered guests. Developed the modules for interacting with the database using JDBC and wrote stored procedures to dynamically generate invoices and find a student a suitable roommate.
    • Simple DB (Java. Team of 4)
      Improved and optimized SimpleDB buffer pool search by using HashMap instead of a linear search. Added support for new data types by adding get and set methods - Short, Boolean, Byte, Date.
  • DevOps
    • DevOps Software Delivery Pipeline (JavaScript, Node.js, and many others. Team of 3)
      End to end software delivery pipeline demonstrating the three prime stages of a software development project - Build, Test and Analysis, and Delivery. Used Git and Jenkins for the build process by configuring git hooks. Used Mocha and Istanbul for unit tests and code coverage measurement. For deployment, we developed a tool to automatically provision and configure a machine on AWS. Configured the machine using Ansible. Used Express.js to build a sample web application and demonstrated a canary release using http-proxy and Redis feature flags. As a special milestone, we implemented a doctor monkey and a maintenance monkey by developing tools to periodically check the system for irregularities and fix them in place.
    • Automatic provisioning (JavaScript, Node.js)
      Developed a tool to interact with the REST APIs of AWS and DigitalOcean and automatically provision a virtual machine on these platforms. Configured the machines using Ansible.
    • Test Generation and Coverage (JavaScript, Node.js, Mocha, Istanbul)
    • Advanced Docker (Docker, docker-compose)
  • Data Structures
    • B-Tree (C Programming)
    • Disk based merge sort (C Programming)
    • In-memory indexes and availability lists (C Programming)
